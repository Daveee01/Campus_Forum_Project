# Panduan Pengguna — KampusConnect Forum

**Selamat datang di KampusConnect!** Forum diskusi untuk mahasiswa berbagi ilmu, bertanya, dan berkolaborasi.

## Memulai

### 1. Registrasi & Login
- Klik tombol **Login** di header (pojok kanan atas).
- Pilih tab **Register** untuk membuat akun baru.
- Isi formulir dengan:
  - Email (gunakan email kampus/pribadi)
  - Password (minimal 6 karakter)
  - Username (singkat, tanpa spasi)
  - Nama Lengkap
  - Program Studi
  - Universitas
  - Tahun Angkatan
  - No. Telepon
- Setelah registrasi, Anda akan otomatis login dan profil akan dibuat.

### 2. Menjelajah Forum (Beranda)
- **Feed Utama**: lihat semua postingan terbaru dari mahasiswa lain.
- **Filter Post**: gunakan tombol filter (Semua / Pertanyaan / Diskusi / Project) untuk mempersempit pencarian.
- **Kategori**: sidebar kanan memiliki kategori (General, Programming, Design, Career, Life) — klik untuk melihat topik tertentu.
- Setiap post menampilkan:
  - Judul, preview konten
  - Tipe post (badge warna biru)
  - Kategori/topik
  - Jumlah upvote dan balasan
  - Nama penulis dan waktu posting

### 3. Membuat Post Baru
- Klik tombol **Buat Post** (header, sebelah kanan).
- Isi formulir:
  - **Judul**: ringkas dan jelas (contoh: "Cara deploy React ke Vercel?")
  - **Tipe**: pilih Pertanyaan, Diskusi, atau Project
  - **Topik/Kategori**: pilih kategori relevan (General, Programming, dll.)
  - **Konten**: tulis detail pertanyaan/diskusi Anda; bisa pakai markdown untuk formatting.
- Klik **Posting** — post Anda akan muncul di feed dan dapat dilihat pengguna lain secara realtime.

### 4. Berkomentar di Post
- Buka post dengan mengklik judul atau card post.
- Scroll ke bagian **Komentar**.
- Tulis komentar di textarea dan klik **Kirim**.
- Komentar akan langsung muncul di bawah post (realtime) dan pengguna lain bisa melihatnya.
- Anda harus login untuk berkomentar.

### 5. Mengelola Profil
- Klik avatar Anda (pojok kanan atas) → pilih **Profil**.
- Di halaman profil, Anda bisa:
  - Lihat data mahasiswa Anda (nama, email, universitas, prodi, tahun, telepon)
  - **Edit Profile**: klik tombol Edit Profile untuk:
    - Ubah bio
    - Ubah foto profil (upload gambar lokal)
    - Perbarui data mahasiswa (nama, username, universitas, dll.)
  - Klik **Simpan** untuk menyimpan perubahan.
- Foto profil yang diupload akan disimpan dan tampil di avatar Anda di seluruh forum.

### 6. Logout
- Klik avatar → pilih **Logout**.
- Sesi akan berakhir dan Anda akan dikembalikan ke halaman beranda (tanpa login).

---

## Tips Penggunaan

- **Gunakan username yang profesional**: hindari username yang terlalu panjang atau tidak pantas.
- **Jaga privasi**: jangan membagikan data sensitif seperti NIM, password, atau alamat rumah di bio atau postingan.
- **Hormati pengguna lain**: gunakan bahasa yang sopan dan konstruktif dalam diskusi.
- **Cari dulu sebelum bertanya**: cek feed atau kategori untuk melihat apakah pertanyaan serupa sudah pernah dibahas.
- **Upvote post/komentar yang bermanfaat**: fitur voting akan segera aktif untuk membantu konten berkualitas lebih terlihat.

## FAQ

**Q: Apakah data saya aman?**  
A: Data Anda disimpan di Firestore (Firebase) dengan enkripsi dan aturan keamanan. Kami tidak menyimpan password plaintext.

**Q: Bisakah saya menghapus post atau komentar saya?**  
A: Fitur hapus post/komentar sedang dalam pengembangan. Sementara hubungi admin jika perlu menghapus konten.

**Q: Bagaimana cara mencari post tertentu?**  
A: Gunakan filter dan kategori di sidebar. Fitur pencarian (search bar) akan ditambahkan di versi mendatang.

**Q: Apakah saya bisa mengedit post setelah diterbitkan?**  
A: Fitur edit post akan ditambahkan. Untuk saat ini, post tidak bisa diedit setelah dipublikasikan.
