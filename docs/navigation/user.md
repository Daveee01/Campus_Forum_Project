# Panduan Navigasi — User

Tujuan: panduan singkat untuk pengguna umum.

- Beranda: `Home` — lihat feed pertanyaan dan diskusi.
- Buat Post: `Create Post` — buat pertanyaan atau diskusi (harus login).
- Profil: `Profile` — lengkapi data mahasiswa saat pertama kali mendaftar.
- Komentar: untuk berinteraksi di tiap posting, klik posting lalu tambahkan komentar (harus login).

Tips:
- Gunakan username yang singkat dan profesional.
- Jaga privasi: jangan memasukkan data sensitif di bio atau posting.
